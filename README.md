## Quick start

1.  Clone this repo using `git clone https://github.com/mohamedsamara/esbuild-react.git`
2.  Move to the directory: `cd <PROJECT_NAME>`.<br />
3.  Run `yarn install` in order to install dependencies.<br />

## Run

```
$ yarn build
```
